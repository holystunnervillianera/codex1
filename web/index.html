    Mac seeding

    First vault import from multiple Apple macOS machines

    Use one visual cloud vault with many owner-controlled Macs. Each Mac runs the same local importer against its own Proton Drive, export, or folder source every 5 or 10 minutes.

    1

    Prepare each Mac

    Install Node 20+, clone/copy this vault repo, and store secrets in `~/.sovereign-vault/device.env`.

    2

    Pick source folders

    Point each Mac at Proton Drive sync, exported Mail attachments, Photos exports, or local archive folders.

    3

    Enable LaunchAgent

    Use `deploy/macos/install-launchagent.sh` to run 5-minute seeding jobs from every Mac into the same encrypted cloud vault.

    bash deploy/macos/install-launchagent.sh \
  --source "$HOME/Proton Drive/VaultInbox" \
  --interval-minutes 5
